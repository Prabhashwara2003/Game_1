# AI Pong Game in C++

A simple Pong game made in C++ where the player can play against the computer.

## Features

- Classic Pong gameplay
- Player-controlled paddle
- Computer-controlled opponent paddle
- Ball movement and collision detection
- Score tracking
- Simple and clean 2D interface
- Basic game loop implementation

## How to Play

- Control your paddle using the keyboard
- Hit the ball back to the computer paddle
- If the opponent misses the ball, you score a point
- If you miss the ball, the computer scores

## Controls

-  W and S  or custom keys based on your implementation
- Close the window to exit the game
